Folder name: Reports

Folder purpose:
This folder contains AI_OS reports, checkpoints, daily summaries, metrics, and report storage policy materials.

What belongs here:
- Daily reports.
- Checkpoints.
- Curated report summaries.
- Report metrics and report storage policy notes.

What does not belong here:
- Active app or service source code.
- Raw source inputs unless a report explicitly embeds verified evidence.
- Root source-of-truth files.
- Trading execution logic, broker credentials, API keys, or live trading controls.

Protected-file warning:
Reports may be historical evidence. Do not edit, move, rename, delete, overwrite, or reformat existing report files without separate human approval and a backup/checkpoint.
