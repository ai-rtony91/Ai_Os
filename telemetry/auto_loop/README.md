# AI_OS Auto-Loop Telemetry

This folder holds generated evidence from the supervised auto-loop DRY_RUN bridge.

Files here are not source-of-truth authority. They are reports, examples, and review evidence for the operator and future validators.

Allowed use:

- demo goals
- DRY_RUN reports
- generated packet, approval, validator, and commit-package evidence

Blocked use:

- secrets
- credentials
- broker execution
- live trading
- active approval inbox state
- active command queue state
- active worker registry state
- runtime process state
