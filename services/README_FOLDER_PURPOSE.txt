Folder name: services

Folder purpose:
This folder contains backend services that support AI_OS orchestration, approvals, workflow state, and audit behavior.

What belongs here:
- Service source code.
- Service package files and configuration.
- Service-level README files.
- Orchestrator implementation materials.

What does not belong here:
- User-facing dashboard app code.
- Reports, checkpoints, or daily summaries.
- Raw source artifacts.
- Broker order placement, live trading enablement, credentials, API keys, or secret material.
