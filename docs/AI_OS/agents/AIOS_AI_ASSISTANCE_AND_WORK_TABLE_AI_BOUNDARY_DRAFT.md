# AI_OS AI Assistance and Work Table AI Boundary Draft

Status: DRAFT
Phase: Phase 12
Stage: 12.19 - Work Table AI Foundation

## Purpose

Document the boundary between AI Assistance and Work Table AI.

## AI Assistance

AI Assistance is the operator helper layer.

Examples:

- explain project status
- summarize checkpoint
- explain command
- remind about safety
- explain next safe action

## Work Table AI

Work Table AI is the work table card and row intelligence layer.

Examples:

- interpret card status
- score work cards
- recommend next card to review
- suggest sorting or filtering
- flag blocked cards

## Boundary Rule

If the feature helps the operator understand the overall project or command context, it belongs to AI Assistance. If the feature scores, sorts, filters, or recommends work table cards or rows, it belongs to Work Table AI.
