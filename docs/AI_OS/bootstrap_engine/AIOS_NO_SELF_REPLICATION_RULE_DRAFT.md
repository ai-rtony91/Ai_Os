# AIOS No Self Replication Rule Draft

Status: Draft planning doc
Stage: 12.7

## Purpose

Define the rule blocking AI_OS from self-replicating into other projects.

## Rule

AI_OS may propose scaffold patterns only after DRY_RUN and human approval. It must not copy itself, bootstrap itself into another repo, or create autonomous project replicas.

## Boundary

No self-replication logic is created.
