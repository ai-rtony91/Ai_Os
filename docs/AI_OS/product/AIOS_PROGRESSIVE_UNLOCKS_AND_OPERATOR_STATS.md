# AI_OS Progressive Unlocks and Operator Stats

AI_OS is an AI-guided developer workspace for non-technical users.

Advanced tools unlock by telemetry and progression, not by cluttering the UI.

## User Progression Levels

Beginner -> Guided -> Intermediate -> Builder -> Advanced -> Experimental -> Developer

## Optional Account Setup

Account setup may be optional for:

- GitHub
- Cloudflare
- Azure
- VS Code

## Unlock Rule

Actionable advanced steps stay locked until safe progress is shown.

## Operator Stats

Telemetry should be shown like operator/game stats:

- Operator Level
- Safety Score
- DRY_RUN Success Rate
- Module Mastery
- Trading Lab Confidence
- Deployment Readiness

Stats must be real usage telemetry, not fake points.

## Product Goal

The end user always knows where to click next.
