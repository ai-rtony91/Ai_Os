# AIOS_OPERATOR_QUICKSTART

## PURPOSE
Beginner-safe quickstart for continuing AI_OS work.

## DEFAULT REPO
C:\Dev\Ai.Os

## FIRST POWERSHELL CHECK
Set-Location "C:\Dev\Ai.Os"
git status --short --branch

## NORMAL WORKFLOW
1. Verify repo path.
2. Verify git status.
3. Create or inspect files.
4. Run validator if available.
5. Review output.
6. Commit safe changes.
7. Push to GitHub.
8. Confirm final clean status.

## STOP CONDITIONS
Stop and ask for review if:
- protected root file would be edited
- delete/overwrite/rename is proposed
- APPLY mode is requested
- startup automation would be enabled
- telemetry persistence would be enabled
- live trading/broker path would be changed
