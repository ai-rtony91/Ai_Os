# AIOS Stage 3 Safe Zones

## Purpose

This stage creates organizer folders only.

## Safety

No files were moved.
No files were deleted.
No files were renamed.
No existing files were overwritten.

## Folder Use

00_START_HERE = first file to open when restarting AI_OS work.
01_CURRENT_STATE = current project checkpoint and working status.
02_RULES = durable operating rules for ChatGPT, Codex, Claude, and local scripts.
03_CONTEXT_PACK = daily drag/drop AI context files.
04_INVENTORY = raw inventory records.
05_CLASSIFICATION = reviewed CSV classification plans.
06_APPROVALS = human-approved action plans before scripts execute changes.
07_MOVE_PLANS = generated move/merge/rename plans.
08_ARCHIVE = preserved older material.
09_REPORTS = daily reports, AARs, progress reports.
10_BRANDING_LATER = logos, icons, splash screen ideas, insignias.
_INBOX_REVIEW = unsorted AI_OS-related files waiting for review.
_DO_NOT_TOUCH = sensitive files or files blocked from automation.
_DELETE_CANDIDATES_HOLD = files considered for deletion only after backup and approval.

## Rule

Move/delete scripts are blocked until an approval file exists in 06_APPROVALS.
