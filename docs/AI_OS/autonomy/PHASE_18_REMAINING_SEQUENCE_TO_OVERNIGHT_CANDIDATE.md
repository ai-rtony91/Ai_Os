# Phase 18 Remaining Sequence to Overnight Candidate

1. Commit and review the 18.13-18.18 runway docs/schemas/fixtures.
2. Draft Phase 18.13 preview packet.
3. Validate Phase 18.13 preview packet through dispatcher hardening.
4. Draft Phase 18.14 corrected Paper SOS resume packet.
5. Only after review, create a future Phase 18.15 short controlled-run executable packet.
6. Review 18.15 output through Phase 18.16 red-team/failure review.
7. If clean, prepare Phase 18.17 longer supervised run.
8. If clean, evaluate Phase 18.18 full overnight candidate criteria.

No stage authorizes runtime by itself.
