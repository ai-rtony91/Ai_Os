# AIOS Codex‚ÜîChatGPT‚ÜîPowerShell Relay (V1)

## Why this exists

This relay removes ad-hoc clipboard interpretation between:

- Codex final packet report
- ChatGPT safety review
- Anthony‚Äôs manual execution step

It standardizes the handoff while keeping **all execution manual** and in Anthony‚Äôs control.

## Handoff flow

1. Codex final report is saved by `New-AiOsCodexReportRelayItem.DRY_RUN.ps1`.
2. ChatGPT prompt is generated by `Invoke-AiOsCodexChatGptRelay.DRY_RUN.ps1`.
3. ChatGPT returns a reviewed PowerShell block.
4. Approved block is saved by `New-AiOsChatGptPastebackItem.DRY_RUN.ps1`.
5. Relay can print next action when pasteback is present.

## Safety boundaries

- All writes are scoped to `control/review_bridge/**`.
- No runtime/queue/lock/approval inbox mutation by this relay.
- `Pasteback` safety scan rejects:
  - `git push --force` / `-f`
  - `git add .`
  - obvious secret patterns
  - direct paths touching runtime/queue/locks/daemon/scheduler/telemetry/dashboard/approval data
- No background execution, scheduler launch, or auto-apply.

## Script matrix

- `automation/orchestration/review_bridge/New-AiOsCodexReportRelayItem.DRY_RUN.ps1`
  - Writes Codex report artifact in `DRY_RUN`/`APPLY` mode.
- `automation/orchestration/review_bridge/Invoke-AiOsCodexChatGptRelay.DRY_RUN.ps1`
  - Reads latest relay item, emits exact prompt text and required review phrase:
  `ChatGPT must review this Codex report and return one PowerShell block only.`
- `automation/orchestration/review_bridge/New-AiOsChatGptPastebackItem.DRY_RUN.ps1`
  - Validates reviewed block and writes artifact in `APPLY` mode.

## Human approval points

- Anthony stores Codex report and pasteback artifacts.
- ChatGPT must explicitly return one PowerShell block.
- Anthony must manually paste and run reviewed commands.
- No automatic merge or automatic command execution is introduced.

## What remains manual

- Exact final decision for each reviewed command
- Merge approval gates in reviewed PR lifecycle scripts
- Final workflow execution in PowerShell

## Operator surface

  operator command and prints the exact next action.
