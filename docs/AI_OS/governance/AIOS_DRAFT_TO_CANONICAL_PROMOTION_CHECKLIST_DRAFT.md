# AIOS Draft To Canonical Promotion Checklist Draft

Status: Draft planning doc
Stage: 12.1

## Purpose

Define the checklist required before any DRAFT document can be promoted to a canonical source of truth.

## Checklist

- DRY_RUN report exists.
- Checkpoint exists.
- Candidate file path is verified.
- Duplicate/conflict review is complete.
- Scope owner is identified or UNKNOWN.
- Protected action status is documented.
- Human approval is explicit.
- Rollback or recovery note exists.

## Stop Conditions

- Missing approval.
- MISMATCH evidence.
- INVALID DATA.
- Unknown critical ownership.
- Protected file impact.

## Boundary

No DRAFT file is promoted by this checklist.
