# AIOS Canonical Conflict Review Draft

Status: Draft planning doc
Stage: 12.1

## Purpose

Plan review rules for conflicting canonical candidates.

## Conflict Types

- duplicate source-of-truth claims
- overlapping scope
- stale roadmap claim
- conflicting safety rule
- mismatched report evidence

## Review Rules

- Mark unresolved conflicts as MISMATCH.
- Mark unverified claims as INVALID DATA.
- Keep missing facts as UNKNOWN.
- Do not merge, rename, or delete files during review.

## Boundary

Conflict review is report-only until a future approved APPLY.
