AI_OS DAILY CONTEXT PACK — USE CASE NOTE

PURPOSE:
This folder stores the daily startup context file used to re-orient ChatGPT, Codex, or another AI assistant at the beginning of an AI_OS work session.

MAIN USE:
Drag and drop 00_AI_OS_DAILY_CONTEXT_PROMPT.txt into ChatGPT when starting a new session so the AI understands:
- the AI_OS project mission
- the current work rules
- the file/folder separation rules
- what actions are blocked
- what requires approval
- where reports, policies, prompts, and logs belong

WHAT THIS FOLDER IS FOR:
- Startup context prompts
- Daily AI orientation notes
- Current project-state summaries
- Safe-session reminder files
- Files that help AI continue work without guessing

WHAT THIS FOLDER IS NOT FOR:
- Daily progress reports
- Error logs
- AAR reports
- Source logs
- Python scripts
- PowerShell scripts
- Broker/API keys
- Trading execution files
- Backtest results

IMPORTANT DISTINCTION:
DAILY_CONTEXT_PACK is a startup/orientation folder.

Reports\daily is where actual daily work reports go.

docs\AI_OS\Prompts is where reusable task prompts go.

docs\AI_OS\policies is where permanent rules go.

HOW TO USE:
1. Open this folder.
2. Drag 00_AI_OS_DAILY_CONTEXT_PROMPT.txt into ChatGPT.
3. Type: Continue AI_OS from latest checkpoint. One safe action first.
4. Wait for ChatGPT to identify the next safe action.
5. Do not upload unnecessary files unless requested.
