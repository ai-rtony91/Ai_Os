# AI_OS Dispatch Human Approval Thresholds

Human approval is required before protected, risky, live, runtime, or external actions.

## Always Requires Human Approval

- APPLY implementation.
- `git add`, commit, push, PR creation, merge, rebase, force push.
- Night Supervisor runtime start.
- Paper SOS resume.
- OpenAI live API call.
- Promptfoo install or execution.
- Computer-use click, type, submit, delete, payment, trading, or destructive action.
- Skill execution when the Skill is unreviewed or over-scoped.
- MCP/tool connector permission expansion.
- Broker/OANDA/live trading.
- Pi GPIO/motor action.

## Approval Does Not Transfer

Approval for one route does not authorize a later route. Validator PASS is evidence only. It is not approval.
