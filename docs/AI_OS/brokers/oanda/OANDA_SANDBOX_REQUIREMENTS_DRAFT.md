# OANDA Sandbox Requirements Draft

## Purpose

This draft lists future review requirements for any OANDA sandbox discussion. It is documentation only.

## Future Sandbox Review Requirements

Before any sandbox connection is even considered, AI_OS would need:

- explicit human approval
- legal/compliance placeholder review
- risk policy review
- telemetry privacy review
- credential handling plan
- `.env` boundary plan
- rollback plan
- audit logging plan
- isolated environment plan
- paper/practice distinction documented

## Still Blocked

This draft does not approve:

- sandbox API calls
- practice account login
- credential access
- token storage
- `.env` edits
- broker adapter code
- paper trading
- live trading
- order placement
