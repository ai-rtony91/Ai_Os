# AI_OS Stage 94 Operator Workflow Escalation Draft

## Purpose

This draft reviews operator workflow and escalation paths. Operator workflow does not approve live automation. Production readiness is not approved by this draft.

No protected root files are edited by this draft. Human approval is required before APPLY actions or protected changes. This draft creates no live automation and no trading automation. LLMs must not be placed in the live order path.

## Workflow Requirements

- Beginner-readable command instructions.
- Exact app/window target.
- Exact repo path.
- Mode declaration.
- Allowed modification list.
- Blocked action list.
- Expected result.
- Stop condition.
- Screenshot/output handoff.
- Escalation when validator fails.
- Escalation when git/network fails.

## Boundary

This draft does not approve production readiness, active automation, startup tasks, active writers, persistence, broker integration, or trading automation.
