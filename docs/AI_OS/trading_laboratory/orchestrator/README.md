# Trading Lab Paper Bot Orchestrator

This folder is the command center for building the AI_OS Trading Lab paper bot.

The orchestrator does not trade. It organizes safe work through shared files, agent roles, task queues, gap logs, validation reports, compact summaries, and next-action files.

Mode: paper-only / simulation-only
