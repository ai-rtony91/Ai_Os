# AI_OS Operator Workbench 001

Stage 16.2 creates the first interactive operator cockpit for AI_OS Trading Lab.

The workbench turns static intelligence visibility into an operator-readable workflow. It remains paper-only and does not expose execution controls, broker controls, autonomous controls, or network order routing.

## Purpose

The operator workbench shows the minimum useful state first:

- Top status strip.
- Replay timeline.
- Confidence timeline.
- Portfolio heat panel.
- Macro-risk overlay.
- Chaos alert surface.
- Replay scrubber placeholder.
- Edge survivability visibility.
- Next Safe Action surface.
