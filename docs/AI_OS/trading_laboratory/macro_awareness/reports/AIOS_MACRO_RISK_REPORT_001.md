# AI_OS Macro Risk Report 001

## Summary

The macro-awareness layer identifies event and market-context risk before paper trading intelligence increases confidence. It is risk-awareness only.

## Event Risk

The layer uses placeholders for CPI, FOMC, NFP, ECB, BOJ, session overlap, pre-event caution windows, and post-event stabilization windows.

## Risk Reactions

- Reduce confidence.
- Freeze confidence increases.
- Reduce simulated size.
- Reduce portfolio exposure.
- Suppress unstable pair ranking.
- Activate WATCH state.
- Activate LOCKDOWN during severe instability.

## Confidence Suppression

Macro uncertainty can suppress confidence but cannot trigger execution. A signal can be marked lower quality during event pressure, volatility expansion, spread instability, or liquidity stress.

## Safety Boundary

Macro awareness informs risk. It does not directly execute trades. No LLM is in a direct live order path. No autonomous headline trading is allowed. Execution isolation remains permanent.
