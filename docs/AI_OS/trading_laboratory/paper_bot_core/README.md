# Paper Bot Core

Paper Bot Core is the first AI_OS paper trading system foundation.

It uses local paper-trade sample data. It reviews Paper Trade Signals, checks Paper Risk Gate rules, creates Paper Decisions only, records Paper Trade Results only, updates Paper Scorecards only, and gives the Next Safe Action.

It does not place real trades. It does not connect to a broker, OANDA, MT5, TradingView, TradersPost, APIs, credentials, account logins, real webhooks, real orders, or real accounts.

## Flow

Paper Trade Signal -> Paper Regime Review -> Paper Risk Gate -> Paper Decision -> Paper Trade Result -> Paper Scorecard -> Next Safe Action
