# Paper Bot Core

Paper Bot Core is the first AI_OS Trading Bot foundation.

It is paper-only. It reviews mock signals, runs a mock risk gate, creates a paper decision, records a simulated paper result, updates a scorecard, and gives the next safe action.

It does not connect to a broker, OANDA, MT5, TradingView, TradersPost, APIs, credentials, webhooks, or real accounts.

## Flow

Signal Intake -> Latency Check -> Regime Tag -> Risk Gate -> Paper Decision -> Paper Trade Result -> Scorecard -> Next Safe Action
