# Phase 18.16 Controlled Run Review Template

Review the controlled run for:

- cwd correctness
- Python `Path.cwd()` correctness
- git root correctness
- locks
- dirty tree
- forbidden path touch
- runtime drift
- loop count
- unexpected writes
- missing stop point
- red-team blocker categories
- prompt injection risk
- secret risk
- OpenAI call risk
- trading/broker/Pi risk
- profitability priority preservation

The run cannot graduate unless all findings are resolved or explicitly accepted by the human owner.
