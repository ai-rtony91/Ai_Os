# AIOS Azure Deployment Boundary Draft

Status: Draft scaffold

## Purpose

Define the boundary for any future Azure deployment planning.

## Allowed Planning Topics

- Target environment inventory.
- Local-first packaging requirements.
- Deployment prerequisites.
- Rollback planning.
- Observability requirements.
- Auth/SSO placeholders.

## Blocked Actions

- Creating Azure resources.
- Editing subscriptions.
- Adding credentials or secrets.
- Deploying AI_OS.
- Enabling public endpoints.
