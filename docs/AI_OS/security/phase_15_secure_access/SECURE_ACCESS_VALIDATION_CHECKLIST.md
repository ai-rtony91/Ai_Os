# Secure Access Validation Checklist

## Required File Checks

- Phase 15 foundation document exists.
- Stage 15.1 front door document exists.
- Cloudflare Access setup plan exists.
- Microsoft Entra SSO setup plan exists.
- YubiKey/passkey operator model exists.
- AI_OS portal zone model exists.
- Validation checklist exists.
- File index exists.

## Forbidden Content Checks

Docs must not contain secret-like assignment strings, private key material, or trading execution activation terms.

## Trading Safety Checks

- Trading Lab / Forex remains in the default paper/simulation or approved demo-review stage unless separately governed.
- No broker login.
- No live orders.
- No OANDA/Webull.
- Secure access does not enable trading execution.

## Account Safety Checks

- No live config changes.
- No installs.
- No account changes.
