# AI_OS Portal Zone Model

## Zones

The protected AI_OS portal should contain:

- Home Portal
- Trading Lab
- Work Table
- Personal Apps
- Admin Zone

## Home Portal

Home Portal is the first page after successful Cloudflare Access, Microsoft Entra SSO, and strong factor checks.

## Trading Lab

Trading Lab remains paper-only.

No broker login, live orders, OANDA, Webull, or trading execution is enabled by secure access.

## Work Table

Work Table remains the planning and operator workflow zone.

## Personal Apps

Personal Apps stay behind the same protected access boundary.

## Admin Zone

Admin Zone requires stronger re-check later before administrative or dangerous actions.
