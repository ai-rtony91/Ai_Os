# YubiKey Passkey Operator Model

## Purpose

YubiKey or FIDO2 passkey provides strong phishing-resistant operator login.

## Planned Role

- Strong factor during login where policy requires it.
- Stronger re-check before Admin Zone actions.
- Stronger re-check before dangerous future actions.

## Operator Meaning

The operator proves possession of a physical key or passkey-capable authenticator before sensitive access is granted.

## Boundary

This document does not register keys, change accounts, install tools, or configure a live identity provider.
