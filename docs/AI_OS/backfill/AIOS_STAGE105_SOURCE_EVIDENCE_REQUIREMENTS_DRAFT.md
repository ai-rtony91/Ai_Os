## Purpose

This draft defines source evidence requirements for future backfill.

No protected root files are edited by this draft. Human approval is required before backfill APPLY. This draft creates no live automation, no backfill performed, no draft promoted, and no trading automation.

- Validator outputs.
- Existing repo files.

## Blocked Evidence
