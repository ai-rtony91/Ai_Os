# Repair Scope Template

Use this template before APPLY.

## Problem

What is wrong?

## Example

Dock player icon shows the wrong playback state.

## Category

UI-only, logic-only, mock-data-only, Trading-Lab-only, connector/API-only, validator gap, or mixed scope.

## Owner

Name the owner or agent.

## Agent Route

Major agent, minor agent, UI readability role, or validator gap role.

## Likely Affected Files

List exact files or folders.

## Blocked Files

List files or folders that must not be touched.

## Validator

List checks that must run.

## Repair Type

One isolated type only.

## Rollback Note

How should the change be backed out if it is wrong?

## Commit Package

Name the one focused commit package.

## Change-Control Link

Use `docs/AI_OS/change_control/CHANGE_REQUEST_TEMPLATE.md` to convert this repair scope into an APPLY request.
