AI_OS HALLUCINATION PREVENTION RULES

1. If the file was not opened, say NOT VERIFIED.
2. If the folder was not scanned, say NOT SCANNED.
3. If the source was not checked, say SOURCE NEEDED.
4. If the command was not run, say COMMAND NOT RUN.
5. If guessing from a screenshot, say SCREENSHOT-BASED.
6. If current software/pricing/API rules matter, search official docs first.
7. If trading/live money is involved, require human approval.
8. If Windows settings are involved, create restore point or backup first.
9. If code is generated, run syntax/test checks before calling it working.
10. If unsure, stop and ask instead of filling the gap.
