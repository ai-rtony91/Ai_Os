# AIOS No-Touch Rules

## Do Not Touch Without Approval

- .env files
- API key files
- broker credentials
- password manager exports
- Windows system folders
- AppData folders
- Git internals
- node_modules
- venv or .venv
- backup archives
- unknown duplicate project folders
- full system image backups

## Blocked Actions

- Mass delete
- Mass overwrite
- Blind merge
- Moving files into GitHub without review
- Editing production scripts without backup
- Changing Windows startup/login behavior without rollback plan
