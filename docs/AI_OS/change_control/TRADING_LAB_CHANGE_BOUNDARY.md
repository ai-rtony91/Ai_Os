# Trading Lab Change Boundary

Trading Lab is the paper-only research and validation area for trading-related work.

Allowed by default in approved scopes:

- paper-only docs
- mock signals
- mock risk records
- mock scorecards
- paper result examples
- validation plans
- orchestrator task and gap records

Blocked by default:

- live trading
- broker routing
- OANDA execution
- API keys
- secrets
- real webhooks
- real orders
- account login
- automatic trade placement
- background trading
- scheduled trading

Trading Lab commits should keep docs, mock data, validators, and executable app code in separate packages.

No profit guarantee is allowed.
