# SYSTEM_LEVEL_AI_WIZARDS.md

## Mission

Build local system-level AI wizards on the OMEN desktop.

The AI_OS controls approved workflows for files, reports, scripts, browser configuration planning, project documentation, and later trading-bot development.

## Current Scope

Version: v0.1

This version is local-first and permission-gated.

## Wizard Layers

Level 1: File, folder, report, and project-tree operations.

Level 2: Approved PowerShell tools and Task Scheduler planning.

Level 3: Desktop UI automation through approved Power Automate Desktop flows.

Level 4: Windows and browser policy preparation.

Level 5: High-risk actions blocked unless manually approved.

## Protected Actions

- BitLocker
- BIOS / UEFI
- Registry
- Broker API keys
- Live trade execution
- Firewall/security rules
- File deletion
- Credential changes

## Core Rule

The system may prepare and recommend. It must not silently execute protected actions.
