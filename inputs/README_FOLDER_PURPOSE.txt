Folder name: inputs

Folder purpose:
This folder contains input materials used to seed or support AI_OS documentation and planning.

What belongs here:
- Raw or lightly prepared input notes.
- Human-provided source text intended for review.
- Input files that may later inform docs or reports after approval.

What does not belong here:
- Final root source-of-truth files.
- Generated reports or checkpoints.
- Internal evidence artifacts that should be preserved under internal.
- Trading execution logic, broker credentials, API keys, or live trading controls.

Protected-file warning:
