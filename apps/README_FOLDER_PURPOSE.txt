Folder name: apps

Folder purpose:
This folder contains user-facing AI_OS applications and interface projects.

What belongs here:
- Dashboard and UI application source code.
- App-specific package files and static assets.
- App-level README files and app configuration.

What does not belong here:
- Root source-of-truth documentation.
- Reports, checkpoints, or daily summaries.
- Raw input evidence or internal source artifacts.
- Trading execution logic, broker credentials, API keys, or live trading controls.

Protected-file warning:
Do not edit, move, rename, delete, overwrite, or reformat protected project files from this folder. Any protected-file action requires separate human approval and a backup/checkpoint.
