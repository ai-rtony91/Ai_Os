# AIOS Dashboard Symbol Assets

These SVG files are original AIOS dashboard assets created for the local dashboard UI.

- No external assets were downloaded.
- No third-party icon pack, native emoji glyph, country flag, broker logo, exchange logo, or brand mark is used.
- Native emoji are secondary only and should not define the dashboard visual identity.
- Every icon must be paired with accessible text or an `AiosSymbol` label.
- Pair icons use abstract market lanes and readable pair text, not flags or official financial marks.
- Trade metadata icons use ledger, ticket, sequence, market state, evidence, replay, reconciliation, risk, and lifecycle motifs only.
- Capital flow icons use alias buckets, gates, sweep lanes, resupply lanes, and compounding target motifs only.
- Money cockpit icons use display gauges, balance ladders, and risk bankroll shields only.
- Future UE5, 3D, or expanded dimensional assets should derive from these motifs without replacing dashboard truth.

The dashboard source of truth remains text status and evidence. Symbols improve scan speed only; they do not create runtime, broker, login, or trading authority.
