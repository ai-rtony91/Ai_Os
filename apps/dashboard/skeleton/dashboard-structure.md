# Dashboard Structure Reference

## Working File

`apps/dashboard/AIOS_STATIC_PREVIEW.html` is the current working dashboard.

## Skeleton Role

This skeleton defines the preferred structure for future dashboard work. It is a design standard/reference only.

## Core Areas

- Sidebar navigation.
- Top status bar.
- Central command ring/tube visual area.
- Dashboard cards layered around the center.
- Floating AI Assistant button.
- Chat drawer mockup for ChatGPT and Codex lanes.
- Tab sections:
  - AI_OS Node
  - Trading Engine
  - Trading Bot
  - Reports
  - Telemetry
  - Admin

## Future Folder Shape

- `concepts/`: experimental UI concepts.
- `styles/`: split-out CSS when approved.
- `scripts/`: local UI-only JavaScript when approved.
- `mock-data/`: static placeholder data when approved.

## Boundary

This file does not approve splitting the current dashboard yet. Future restructuring requires explicit approval and validation.
