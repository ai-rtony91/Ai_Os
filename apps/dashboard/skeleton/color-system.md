# Dashboard Color System Reference

## Purpose

This file records the AI_OS static dashboard color direction. It is a reference only and does not change the working dashboard.

## Theme Direction

- Dark burgundy and black background.
- Green terminal glow for active command text.
- Muted green for supporting text.
- Red or pink-red for blocked states.
- Amber for review/warning states.

## Usage Rules

- Use green terminal glow for active operator status and command output.
- Use burgundy/black layers for depth and focus.
- Use blocked colors only for safety boundaries and disabled states.
- Keep contrast high enough for operator readability.
- Avoid visual styles that imply live execution or production activation.

## Safety Copy Colors

Safety labels such as no backend, no API calls, no persistence, no broker/trading automation, and no credentials must remain visually prominent.

## Boundary

This color system is not a production brand guide and does not automatically affect `AIOS_STATIC_PREVIEW.html`.
