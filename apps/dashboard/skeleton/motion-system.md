# Dashboard Motion System Reference

## Purpose

This file documents the motion rules for the AI_OS dashboard concept. It is a reference only.

## Approved Motion Concepts

- Subtle parallax background layers.
- Animated grid drift.
- Slow orbit/ring lines.
- Central tube/ring command interface motion.
- Mouse movement visual response.
- Scroll movement visual response.
- Local tab transitions.
- Local chat drawer open/close interaction.

## Motion Limits

- Not headset VR.
- Not overbuilt.
- No motion that suggests live execution.
- No animation tied to real broker, telemetry, report writer, or backend state.
- No network-driven animation.

## Local JavaScript Boundary

JavaScript may be used only for local UI interactions such as tabs, toggles, drawer state, mouse parallax, and scroll parallax.

## Boundary

Motion rules do not activate automation, persistence, backend calls, or trading behavior.
