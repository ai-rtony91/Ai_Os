# AI_OS Dashboard Skeleton Reference

## Purpose

This folder is a design standard/reference for the AI_OS dashboard. It does not run the dashboard and does not automatically affect the working UI.

## Current Working Dashboard

`apps/dashboard/AIOS_STATIC_PREVIEW.html` is the current working dashboard preview.

Do not treat this skeleton folder as a replacement for the working preview. Changes here are reference-only until a future approved APPLY step explicitly updates the dashboard.

## Future Structure

Future dashboard work may split the static preview into:

- `concepts/` for future experiments.
- `styles/` for CSS systems.
- `scripts/` for local UI-only JavaScript.
- `mock-data/` for static placeholder data.

## Safety Boundary

- No backend.
- No API calls.
- No credentials.
- No persistence.
- No trading/broker automation.
- No live order path.
- No startup tasks.

Skeleton files are reference only and do not activate dashboard behavior.
