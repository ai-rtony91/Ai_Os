# TradingView Alert Template Guide

The local template represents the alert fields AI_OS expects to review:

- source
- symbol
- pair
- timeframe
- signal_type
- direction
- price
- strategy_name
- alert_time
- risk_mode
- notes

Required safety value:
- `risk_mode` must be `paper_only`.

This guide is for manual template review only. It does not require TradingView login, API access, or external alert delivery.
