# AI_OS Low-Chaos Operator Model 001

The low-chaos operator model keeps the Trading Lab readable under pressure.

## Operating Rule

Show the operator the next relevant action, visible safety state, current confidence state, edge state, replay state, macro risk, portfolio heat, and any chaos alert. Keep deeper diagnostics collapsed.

## Behavior

- Collapse advanced details by default.
- Preserve existing routing.
- Preserve paper-only status.
- No autoplay behaviors.
- No execution controls.
- No broker controls.
- No clutter explosion.

## Why It Matters

Trading intelligence only helps when the operator can quickly understand what AI_OS trusts, what it distrusts, and why. A noisy interface creates hesitation and hides risk.
