# AI_OS Autonomy Limitations 001

## Purpose

This report defines what AI_OS autonomy cannot do in trading architecture.

## AI_OS May

- analyze signals
- score confluence
- classify regimes
- rank portfolio opportunities
- make adaptive recommendations
- simulate paper routes
- generate telemetry

## AI_OS May Not

- place live orders
- connect to live brokers
- route webhook executions
- deploy itself into execution
- modify execution code automatically
- use real funds
- bypass paper validation

## Permanent Rule

Recommendation-only AI behavior is required. Execution requires separate approval systems and independent controls.
