# Paper-Only Boundary

This orchestrator is paper-only and simulation-only.

Blocked:
- broker execution
- OANDA execution
- API keys
- secrets
- real webhook activation
- real orders
- live market execution
- account login systems
- scheduled trading
- background trading

Allowed:
- mock signal records
- manual paper review
- simulated outcomes
- validation reports
- compact summaries
- next safe action files

Live execution remains BLOCKED.
