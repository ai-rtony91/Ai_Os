# Forex Strategy London Breakout V1

## Purpose

London Breakout V1 is a paper-only strategy concept for reviewing whether a session-based breakout idea has enough confluence for local simulation.

No broker logic, OANDA connection, webhook execution, API keys, external requests, live orders, or commits are part of this file.

## Strategy Idea

The strategy evaluates a potential London-session range break using session alignment, volatility condition, spread condition, and risk gate status.

## Inputs

- Pair from the local forex pair registry
- Timeframe
- Session alignment
- Regime tag
- Volatility condition placeholder
- Spread condition placeholder
- Confidence score
- Reject reason

## Paper-Only Decision Role

This strategy can only produce paper review decisions. Live execution remains BLOCKED.
