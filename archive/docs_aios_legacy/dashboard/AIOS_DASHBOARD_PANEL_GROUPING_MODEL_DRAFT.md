# AI_OS Dashboard Panel Grouping Model Draft

## Purpose

Group dashboard content into clear operational panels so the UI feels organized instead of scattered.

## Panel Groups

Command Center:
- AI_OS current status
- Next safe action
- Latest checkpoint
- Latest commit
- Major stats summary

Current Project / About:
- What AI_OS is
- Current phase/stage
- Project purpose
- Local-first system boundaries

Progress:
- Phase completion
- Stage completion
- Files created
- KB / MB created
- Daily growth

Reports + Telemetry:
- Daily reports
- Checkpoint reports
- Telemetry snapshots
- Progress ledger

Validators:
- Master validator status
- Recent PASS/WARN/FAIL
- Validation queue

Safety:
- No secrets
- No broker/live trading
- Blocked actions
- Approval gates

AI Assistance:
- Assistant status
- Mock assistant output
- Next-action guidance

Work Table AI:
- Row/card intelligence
- Scoring/recommendation placeholder
- Approval-required flags

Apps / Admin:
- App registry
- Tool registry
- Admin/system actions
