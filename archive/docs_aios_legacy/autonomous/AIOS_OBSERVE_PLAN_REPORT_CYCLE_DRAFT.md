# AIOS Observe Plan Report Cycle Draft

Stage: 11.4
Status: Draft planning doc

## Purpose

Define the autonomous operating loop as observe, plan, report, and stop for approval.

## Cycle

1. Observe repository evidence.
2. Plan safe next action.
3. Report files inspected, risks, unknowns, and proposed changes.
4. Stop before APPLY unless approval exists.

## Boundary

The cycle must not perform protected actions automatically.
