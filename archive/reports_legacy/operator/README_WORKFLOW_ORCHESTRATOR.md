# AI_OS Workflow Orchestrator Reports

This folder stores the operator-facing workflow packet generated by:

```powershell
powershell -ExecutionPolicy Bypass -File automation/operator/Invoke-AiOsWorkflowOrchestrator.ps1
```

Primary packet:

```text
Reports/operator/AIOS_OPERATOR_NEXT_ACTION_PACKET.md
```

The packet is DRY_RUN and operator-approved only. It summarizes current repo status, worker lanes, work queue items, approvals, validators, blocked reasons, exact files ready for APPLY or commit, and the next safe action.

The packet does not grant permission to APPLY, commit, push, connect brokers, use secrets, or enable live trading.
