# AI_OS Codex Worker Profile Launcher

Purpose:
Launch Codex workers from named layout profiles.

Supported Profiles:
- LEFT_VERTICAL
- LEFT_2X2

Current Preferred Profile:
LEFT_VERTICAL

Why:
This keeps 4 Codex workers on the left side while leaving center/right space for ChatGPT, dashboard, charts, and operator PowerShell.

Safety:
- No autonomous commits.
- No autonomous pushes.
- No autonomous deploys.
- No broker execution.
- Paper-only trading workflow.
