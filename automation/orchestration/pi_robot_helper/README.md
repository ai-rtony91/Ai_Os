# AI_OS Pi Robot Helper Preview Lane

This directory contains East-owned DRY_RUN-only preview support for a future AI_OS Pi Robot Helper alert assistant.

Boundary:

- preview only
- no hardware
- no GPIO
- no motors
- no camera
- no microphone
- no speaker
- no WOL packet sending
- no service or scheduler registration
- no real TTS/STT
- no repo state mutation
- no commit, push, or merge authority

Omen remains the authority for the operator workstation. The Robot Helper may only display or announce approved status in future separately approved phases. Current files render text previews only and must not drive hardware, send alerts, wake machines, or mutate AI_OS runtime state.
