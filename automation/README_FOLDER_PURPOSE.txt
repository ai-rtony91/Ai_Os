Folder name: automation

Folder purpose:
This folder contains approved AI_OS automation scripts, launchers, reporting helpers, and workflow pipeline materials.

What belongs here:
- Automation pipeline notes.
- PowerShell or script files for approved AI_OS automation.
- Reporting and startup workflow helpers.
- Dry-run-first automation support files.

What does not belong here:
- Protected root source-of-truth files.
- Raw source evidence.
- User-facing app source code.
- Broker order placement, live trading enablement, credentials, API keys, or secret material.

Protected-file warning:
Automation must default to DRY_RUN. Do not edit, move, rename, delete, overwrite, or reformat protected project files without separate human approval and a backup/checkpoint.
